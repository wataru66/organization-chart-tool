üéØ Basic Usage

Step-by-Step Workflow

  1. Load Your Data
    Click the file drop zone or drag & drop your Excel file. The file will be automatically processed and validated.
  2. Configure Display Settings
    Choose your base organization, level limit, font size, and box size. Toggle manager visibility if needed.
  3. Generate the Chart
    Click "Generate Chart" to create your organization chart. The layout will be automatically calculated.
  4. Review and Adjust
    Use the data table editor to make changes, or adjust display settings and regenerate.
  5. Export Your Chart
    Choose from SVG, PNG, HTML, or Excel export formats to save or share your chart.

üêõ Troubleshooting

Common Issues & Solutions

‚ö†Ô∏è "PNG export requires html2canvas library"

Solution: The PNG export feature requires the html2canvas library. Use SVG export instead, or check that the library loaded correctly in your browser's developer console.

‚ö†Ô∏è "File format not supported"

Solutions:

  • Ensure your file has a .xlsx or .xls extension
  • Verify the file is a valid Excel format
  • Try re-saving the file in Excel with "Save As" ‚Üí Excel Workbook
  • Check that the file isn't corrupted

‚ö†Ô∏è "No data loaded"

Solutions:

  • Verify that required columns (Name, Call Name, Role) contain data
  • Check that your Excel file follows the correct column structure
  • Use the "Validate Data" feature to identify specific issues
  • Try loading the sample data first to test the tool

Data Validation Errors

Required Field Errors

  • "Name is required": Fill in the Name column for all rows
  • "Call Name is required": Each organization needs a unique Call Name
  • "Role is required": Specify the role/position for each person

Hierarchy Errors

  • "Parent organization not found": Ensure Parent Org values match existing Call Names exactly
  • "Circular reference detected": Check that organizations don't reference themselves or create loops
  • "Level out of range": Use hierarchy levels between 1-10

Color Format Errors

  • "Invalid color format": Use HEX codes (#2196f3), color names (blue), or palette colors (blue-medium)
  • Colors not appearing: Clear browser cache or try a different color format
